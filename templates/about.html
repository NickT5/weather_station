About Weather Station

Weather Station is a simple web application that provides current weather information for locations around the world.

Features

  • Search for weather by location name
  • View current temperature, conditions, humidity, pressure, and wind information
  • Interactive map showing the location
  • Temperature displayed in both Celsius and Fahrenheit
  • Clean, responsive interface

Technology

This application is built with:

  • Python - Backend programming language
  • Flask - Web framework
  • Open-Meteo API - Free weather data provider (no API key required)
  • Leaflet.js - Interactive maps
  • HTML, CSS - Frontend

How It Works

When you search for a location:

  1. The application uses the Open-Meteo Geocoding API to convert your location name into geographic coordinates (latitude and longitude)
  2. These coordinates are then used to fetch current weather data from the Open-Meteo Forecast API
  3. The weather data is processed and displayed in a user-friendly format

Usage

Simply enter a location in the search box on the home page and click "Get Weather". For best results, include both the city and country (e.g., "Paris, France" or "Tokyo, Japan").
